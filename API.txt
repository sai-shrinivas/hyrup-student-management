List of APIs:-

1. Register User
  POST /api/auth/register

2. Login User
  POST /api/auth/login

3. Get All Users
  GET /api/users
(Requires Token for Authentication)

4. Get Single User by ID
  GET /api/users/:id
(Requires Token for Authentication)

5. Create User (Admin)
  POST /api/users
(Requires Token for Authentication + Admin Role)

6. Update User (Admin)
  PUT /api/users/:id
(Requires Token for Authentication + Admin Role)

7. Delete User (Admin)
  DELETE /api/users/:id
(Requires Token for Authentication + Admin Role)
